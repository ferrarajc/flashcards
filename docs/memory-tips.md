# Memory Tips

These tips rotate through the typewriter prompt on the Learn screen while a card is face-down.
To add a new tip, append it to this list **and** to the `MEMORY_TIPS` array in `screens/LearnScreen.js`.

## Tips

1. If you know the answer, say it out loud
2. Give your brain time to try to recall the answer
3. Sleep after studying helps memories stick
4. Imagine pictures to make ideas more memorable
5. Review harder cards more often than easy ones
6. Study in different places now and then
7. Revisit older cards so they don't fade
8. Keep sessions short so you don't burn out
